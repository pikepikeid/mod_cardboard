## 利用規約
規約はいつも通りこのへんに準拠

## 導入方法
CM3D2\Sybaris\GameData内に適当なフォルダを作成してファイルを全て入れてください。

### 使い方
* アクセサリー/腕にあります、面倒くさいので腕のままです。メイドも消えるVerです。
* 被れません。
* 強化ダンボールじゃないのでLBXバトルにも使えません。
